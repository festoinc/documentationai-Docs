### Usage in this API Documentation

Throughout these docs you'll see these components used for:

- **Callouts** — highlighting important API constraints, deprecations, and tips
- **Steps** — walking through integration flows and setup procedures
- **Tabs** — showing PMS vs PM token types, or daily rates vs LOS pricing
- **Cards** — linking to related API sections and endpoints
- **Expandable** — FAQ answers and troubleshooting details
- **ParamField** — documenting API parameters with type, location, and required status
