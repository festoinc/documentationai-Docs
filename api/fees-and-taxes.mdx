## Overview

Set fees (cleaning, damage waiver, extra bed/person) and taxes for properties. Requires **PM-level credentials**.

<Callout kind="alert">
You must **always send the complete list** of fees and taxes. Any fees or taxes currently in the system that are not included will be **deleted**. Send an empty list to remove all.
</Callout>

<Callout kind="info">
- All taxes are expressed as a **percentage**.
- Security deposits for Airbnb must be between **100–5,000 USD**.
- This is a **full update** — always send the complete list.
</Callout>

## Create Fee and Tax

| Field | Value |
|---|---|
| **Method** | `POST` |
| **Endpoint** | `/taxfee` |
| **Auth** | JWT (PM credentials) |

```bash
curl -X POST "https://apidemo.mybookingpal.com/taxfee?jwt=TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "data": {
      "productId": "1235124634",
      "fees": [
        {
          "name": "Cleaning Fee",
          "type": "FLAT",
          "valueType": "PER_STAY",
          "mandatory": true,
          "value": 75.00,
          "currency": "USD"
        }
      ],
      "taxes": [
        {
          "name": "City Tax",
          "type": "PERCENTAGE",
          "value": 10.00,
          "mandatory": true
        }
      ]
    }
  }'
```

## Get Fee and Tax Mandatory

| Field | Value |
|---|---|
| **Method** | `GET` |
| **Endpoint** | `/validation-settings/get-validation` |
| **Auth** | JWT (PM credentials) |

Returns whether fees and taxes are mandatory for each property.

**Response:**

```json
{
  "data": [
    {
      "validationSettings": [
        { "productId": 1235124634, "isFeeMandatory": false, "isTaxMandatory": false }
      ]
    }
  ]
}
```

## Remove Validation Settings

| Field | Value |
|---|---|
| **Method** | `POST` |
| **Endpoint** | `/validation-settings/removed` |
| **Auth** | JWT (PM credentials) |

Resets property-level fee and tax settings to defaults (both mandatory).

```bash
curl -X POST "https://apidemo.mybookingpal.com/validation-settings/removed?jwt=TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"data": [1235124634]}'
```
